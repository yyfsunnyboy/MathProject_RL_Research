# 高中科展：從「精準測驗」到「智慧教學」—— 結合強化學習與心理學的適性化學習系統 (Adaptive Learning System)

本專案旨在重新定義教育軟體中的「適性化測試與教學」，從傳統的「找出缺點」推進到「陪伴學習成長」，透過量化教育心理學理論並導入強大的人工智慧演算法，打造出更貼近人性、更具教學成效的模擬教育環境。

---

## 📖 1. 專案背景與痛點 (Problem Statement)

現代主流的教育科技多數依賴 **適性測驗 (Computerized Adaptive Testing, CAT)**，這類系統的核心精神是：**「用最短的時間、最精準地測出學生真實的能力分布（或者說，哪裡不會）。」**
* **運作機制**：CAT 會計算題目對學生的「最大訊息量（Fisher Information）」，不斷挑選難度剛好卡在學生能力邊界、最能將學生「考倒或鑑別出來」的題目。
* **教育痛點**：在真實的教育場域中，學生不是被單向測量的機器。如果系統永遠只給能「精準打擊弱點」的困難題目，學生會感到強烈的挫折感、產生認知超載，進而抗拒學習。
* **我們的解法**：我們觀察到，真正優秀的老師不僅僅是「考試機器」，他們懂得搭建鷹架（Scaffolding）、尋找學生的「最近發展區（ZPD）」，並利用知識點之間的關聯性來引導學生。因此，我們透過本專案，試圖將這些「無形的教學藝術與心理模型」轉化為「可量化的數學模型」，寫進 AI 的大腦裡。

## 🧠 2. 理論基礎與核心思想 (Theoretical Foundations)

本專案的虛擬大腦與環境建立在以下幾個嚴謹的科學理論之上：

1. **項目反應理論 (Item Response Theory, IRT / MIRT)**：
   我們不使用傳統的古典測驗理論（算總分），而是利用 MIRT 數學模型（結合鑑別度 $a$ 與難度 $b$），來描繪學生面對特定題目時的伯努利機率（答對率）。以確保模擬的隨機性符合真實世界數據。
   
2. **最近發展區理論 (Zone of Proximal Development, ZPD, Vygotsky)**：
   人類學習新事物的最佳區間，不是做自己已經滾瓜爛熟的題目（浪費時間），也不是做完全看不懂的題目（認知超載），而是落在答對率約 $60\% \sim 80\%$ 之間、需要稍作努力的區間。我們在程式中使用「高斯分佈曲線」對此進行了數學建模。

3. **知識擴散模型 (Knowledge Diffusion Modeling)**：
   知識不是孤立存在的，比如「要學會解一元一次方程式 (equation-solving)，往往得先懂代入法 (substitution)」。我們透過有向無環圖 (DAG) 的關聯矩陣，模擬了學生在學會一個知識點後，相關聯的知識點也能連帶產生化學變化的過程。

4. **認知超載懲罰 (Cognitive Overload Penalty)**：
   這是專案的一大亮點。當系統給出的題目讓學生的答對勝率低於 $20\%$ 時，不僅學習效益（Reward）微乎其微，甚至會觸發能力倒退（能力值被強行扣除），模擬學生因連續挫折而產生的「習得性無助」。

## 🏗️ 3. 專案架構與程式說明 (Implementation Details)

為了證明我們的假說，本專案實作了兩個互為對照、層層遞進的 Python 實驗腳本，完美呈現了從「測量」到「教學」的演進過程：

### 實驗一：測驗導向模型 —— `20260309-2.py`
* **研究核心**：探索傳統適性測驗 (CAT) 在數學上的極限。
* **運作邏輯**：學生的能力被預設為**靜態不可變**。系統採用「最大後驗機率更新 (MAP)」與貝氏統計中的「共變異數矩陣」，發展出 Baseline, Variance, Trace 等尋找最大 Fisher 訊息量的挑題策略。
* **結果與定位**：這支程式成功證明了適性測驗（如 Trace 策略）能讓系統對學生能力的預估誤差 (RMSE) 呈現最快速的收斂（下降）。它是我們實驗完美的 **「對照基準線 (Baseline)」**。

### 實驗二：教學導向模型（核心亮點）—— `0315RL實驗.py`
* **研究核心**：突破傳統，打造具備同理心與教育理念的 AI 導師 (RL Agent)。
* **運作邏輯**：學生的能力變為**極度動態**。我們利用 OpenAI Gymnasium 將上述的心理學模型 (ZPD, 擴散, 超載) 包裝成一個擬真的「教室環境」，並引入 Stable-Baselines3 的 **近端策略最佳化 (Proximal Policy Optimization, PPO, 強化學習演算法)** 加以訓練。
* **結果與定位**：RL Agent 在訓練後，不再一味追求「資訊量」，而是學會觀察學生的疲累度 (fail_streak) 與知識結構，並為了榨取最大的能力進步回報 (Reward)，而發展出與傳統 CAT 截然不同的「鷹架式挑題法」。這是整個科展的 **「實驗突破點」**。

*(註：本核心程式更使用了 PyTorch 動態擬合 ASSISTments 2017 真實資料集，確保環境的底層難度/能力分佈是基於美國超過數十萬條真實學生的作答紀錄，而非純亂數捏造)*

## 📊 4. 預期結果與科學貢獻 (Expected Results & Contributions)

透過執行最終的成效比較測試 (`RealData_Teaching_Process_Stratified.csv`)，並將學生依能力高低（Low, Mid, High）劃分，我們預期（也證實了）以下關鍵結論：

1. **傳統適性測驗 (CAT) 教不好後段班學生**：
   CAT 為了摸清低程度學生的底線，時常丟出高鑑別度的困難題目，導致學生反覆觸發「認知超載懲罰」，在 30 步的模擬結束後，整體能力幾乎持平甚或倒退。
2. **RL 強化學習導師展現了「人類教育智慧」**：
   受過 PPO 訓練的強化學習模型會自發性地尋找學生的「ZPD（最近發展區）」。對於低程度學生，它會精準投遞難度相仿、答對率偏高的墊腳石題目；對於卡關的知識點，則展現出「迂迴」策略，先教有關聯性 ($DAG$ 矩陣) 的其他弱點，藉由擴散效應帶動學習。
3. **不同程度學生的效益差異**：
   在中高程度學生群體中，CAT 與 RL 導師的成效差距會逐漸縮小；因為這類學生本身對於難題的抗壓性（ZPD 高峰寬度）與基礎（不容易掉入 20% 防禦線以下）較好，這進一步佐證了「因材施教」機制的運作是正確的。

**綜上所述，本科展成功透過程式碼與人工智慧演算法，量化了無形的教育心理學現象，並提出了一個超越市面傳統「適性測驗」、具備高度教學價值的「智慧教學系統 (ITS) 框架」，為未來的數位教育發展提供了強有力的實證基礎。**
